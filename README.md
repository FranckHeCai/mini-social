# Mini-Social

A small-scale social media application inspired by X (formerly Twitter), featuring basic functionalities such as CRUD operations. This project demonstrates a full-stack implementation with a **FastAPI** backend and a **Nuxt.js** frontend.

## Features

- **Create, Read, Update, Delete (CRUD)** posts
- Simple and clean user interface with Nuxt.js
- RESTful API implementation using FastAPI
- Modular and scalable project structure

## Tech Stack

- **Backend:** Python, FastAPI, SQLAlchemy
- **Frontend:** Nuxt.js
- **Database:** PostgreSQL
- **API Testing:** TBD

## Installation

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
Create a virtual environment manualy or use uv
```bash
cd backend
pip install -r requirements.txt
uv run main.py
```
